## Installation

## NodeJS

```
npm install
npm start
```

## Postgres

Install following with link: [postgres](https://github.com/postgres/postgres)

Step Run: 

1. Edit file: `database.module.ts` in `src/database` to config database
2. Run command: `npm install`
3. Run command `npm run start`
